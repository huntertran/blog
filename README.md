Blog
---

My personal blog built with `hexo` and next theme

# Setting up

Install NPM

```s
npm install -g npm
```

Install Hexo CLI

```s
npm install hexo-cli -g
```

Install packages

```s
npm i
```

# commonly uses command

```s
# run local
hexo serve

# new post
hexo new post

# run local server
hexo serve
```
